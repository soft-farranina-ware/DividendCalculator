## Description
This is a mobile Android application developed for an individual assignment. It calculates the dividend from a unit trust investment based on the invested fund amount, annual dividend rate, and duration of investment (months).

## Features
* **Dividend Calculation:** Calculates monthly and total yearly dividends.
* **Validation:** Ensures months do not exceed 12.
* **About Page:** Displays author information and GitHub repository link.
* **Theme:** Custom Brown/Coffee aesthetic.

## Tools Used
* Android Studio (Java)
* XML for Layouts
